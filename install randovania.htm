How to install Randovania
Please be sure that you always use the newest version to ensure the best play experience.
Installer version (Windows only for now) Execute the installer and follow the wizard. Executable version Unpack the zip file and start the application. You need no other programs or tools to start Randovania. To run a multiworld session you will need to install Discord.
